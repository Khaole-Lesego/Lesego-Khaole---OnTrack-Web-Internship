# Micro-Learning Habit Tracker

A beginner-friendly web project built with **HTML**, **CSS**, and **JavaScript**.

This project is a **micro-learning habit tracker** inspired by the structure of platforms like **Duolingo** and **Blinkist**.

The goal is to build a web-based platform that helps users:

- choose a learning goal,
- receive short learning content,
- track daily progress,
- build consistency through streaks,
- and stay motivated through simple gamification.

---

## Project Goal

The final project will be a **fully functioning website** that combines:

- **Onboarding**: collect user goals and interests
- **Lesson Display**: show short learning content
- **Streak Tracking**: track how many days the user returns
- **Gamification**: points, badges, and levels
- **Quiz / Reflection**: test understanding after each lesson
- **Content Library**: organize all lessons in one place
- **Community / Sharing**: allow users to share progress or reflections

---

## Why This Project Exists

This project is meant to solve a real need:

- people want to learn, but they often do not know where to start
- people often quit because the process feels too big
- people need structure, encouragement, and consistency
- short daily learning is easier to maintain than large overwhelming lessons

This website is designed to make learning feel small, clear, and achievable.

---

## Recommended Development Approach

This project should be built in **small mini-projects** first.

Each mini-project should work on its own before being combined into the full platform.

That way, the project stays manageable and easier to understand for a beginner.
